# Site hero / key images

- **`hero-home.jpg`** – Homepage hero (“our building” / main hero). Used on the homepage. Recommended: 2560px wide or larger, high quality JPG. Replace this file to change the building/hero photo.

Other page heroes (solar, electrical, contact, etc.) are still set in `src/config/images.ts`; you can add more files here and switch those to local paths when you’re ready.
